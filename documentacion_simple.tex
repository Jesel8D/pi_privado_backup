\section{Servicios y Puertos}

Backend (NestJS) → Puerto 3001 \\
PostgreSQL → Puerto 5432 \\
MongoDB → Puerto 27017 \\
Nginx (Proxy) → Puerto 80

\section{M\'odulos Backend Implementados}
\begin{enumerate}
\item Auth: Autenticaci\'on JWT con Google OAuth
\item Users: Gesti\'on de usuarios y roles
\item Products: Cat\'alogo de productos
\item Inventory: Control de inventario y lotes
\item Sales: Registro de ventas diarias
\item Orders: Sistema de pedidos
\item Dashboard: Panel de control con m\'etricas
\item Reports: Reportes semanales
\item Break-even: An\'alisis de punto de equilibrio
\item Forecast: Pron\'osticos de demanda
\item Expiration: Alertas de productos pr\'oximos a vencer
\item Audit: Sistema de auditor\'ia con MongoDB
\item Benchmarking: M\'etricas de rendimiento con BigQuery
\end{enumerate}

\section{Tecnolog\'ias Utilizadas}
\textbf{Frontend:} Next.js 14, React 18, TailwindCSS, TypeScript \\
\textbf{Backend:} NestJS, TypeORM, PostgreSQL, MongoDB \\
\textbf{DevOps:} Docker, Docker Compose, Nginx \\
\textbf{Cloud:} Google BigQuery, Google Cloud Storage \\
\textbf{IA:} Sistema SOA Auto-Healing con Ollama
